# **App Name**: SoarScape

## Core Features:

- Obstacle Generation: Dynamically generate obstacles with varying heights and spacing.
- Character Control: Implement simple tap/click controls for character flight.
- Score Tracking: Track and display the player's score based on distance traveled.
- Game Over Detection: Detect collisions with obstacles and trigger game over sequence.
- Procedural Generation of level using tool: Use an AI tool to generate a procedural map for each run based on various parameters (difficulty, style, theme).

## Style Guidelines:

- Primary color: Sky Blue (#87CEEB) to evoke a sense of open space and freedom.
- Background color: Light Blue (#B0E2FF), a desaturated version of the primary, creating a bright, airy feel.
- Accent color: Sunny Yellow (#FFD700) for high-visibility elements and score display.
- Headline and body font: 'Space Grotesk', a sans-serif font for a clean, futuristic look.
- Simple, flat icons for the game's user interface.
- Full-screen layout to maximize the gaming experience.
- Smooth animations for character movement and obstacle scrolling.
